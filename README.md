# Modern academic homepage for GitHub Pages

## Files
- `index.html`
- `assets/style.css`
- `assets/profile.jpg` (placeholder)
- `assets/Manjish_Pal_CV.pdf` (placeholder resume)

## Publish on GitHub Pages
1. Create a public repository named `yourusername.github.io`
2. Upload all files from this folder
3. Replace `assets/profile.jpg` with your real photo
4. Replace `assets/Manjish_Pal_CV.pdf` with your actual CV
5. Commit the files
6. Open `https://yourusername.github.io`

## Notes
- The publication list was prepared from public scholarly sources and can be edited in `index.html`
- You can add sections for teaching, awards, talks, grants, or projects later
